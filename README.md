# 🌐 Network Monitor (Python Polling)

Este projeto é um script simples e eficiente em Python para realizar o monitoramento de disponibilidade de servidores via **Polling**. Criado como parte dos meus estudos de Redes de Computadores e Python.

## 🚀 Funcionalidades
- **Polling System:** Verifica o status de um servidor em intervalos definidos.
- **Error Handling:** Identifica se o servidor está offline ou retornando erros HTTP.
- **Feedback Visual:** Logs coloridos (emulados via texto) no terminal para fácil leitura.

## 🛠️ Tecnologias
- Python 3.x
- Requests Library (HTTP for Humans)
